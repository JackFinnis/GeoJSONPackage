Structures

Bounding​Box

Information on the coordinate range for other GeoJSON objects. Contains two Positions which denote the most southwesterly and most northeasterly points of the object's geometry. See RFC9746 Section 5 for more information.

Feature

A Feature represents a spatially bounded "thing" which is represented by its geometry (or missing it if it is unlocated).

Point

Polygon

A collection of LinearRings. The first ring is expected to be the exterior ring, any others are interior rings. The exterior ring bounds the surface, and the interior rings (if present) bound holes within the surface.

Polygon.​Linear​Ring

A closed LineString containing four or more Positions. The first and last value are expected to be the same so that the ring is closed. A linear ring is the boundary of a surface or the boundary of a hole in a surface and must follow the right-hand rule with respect to the area it bounds, i.e. exterior rings are counterclockwise, and holes are clockwise.

Position

A fundamental geometry construct containing the coordinates of a geographic position and an optional altitude. See RFC9746 Section 3.1.1 and Section 4 for more information.

Typealiases

Geometry​Collection

A heterogeneous composition of other Geometry objects. For example, a Geometry object in the shape of a lowercase roman "i" can be composed of one Point and one LineString. Please try and avoid nested GeometryCollections to maximize interopability. See RFC7946 Section 3.1.8 for more information.
