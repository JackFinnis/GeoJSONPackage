polygon 

case polygon(: Polygon)

A collection of LinearRings. The first ring is expected to be the exterior ring, any others are interior rings. The exterior ring bounds the surface, and the interior rings (if present) bound holes within the surface.

multi​Polygon 

case multiPolygon(: MultiPolygon)

A collection of Polygons.

geometry​Collection 

case geometryCollection(: GeometryCollection)

A heterogeneous composition of other Geometry objects. For example, a Geometry object in the shape of a lowercase roman "i" can be composed of one Point and one LineString. Please try and avoid nested GeometryCollections to maximize interopability. See RFC7946 Section 3.1.8 for more information.
